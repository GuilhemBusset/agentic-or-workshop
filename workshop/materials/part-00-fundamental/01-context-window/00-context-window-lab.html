Lost in the Middle — U-Shaped Attention

LLMs recall information near the beginning and end of context far better than information buried in the middle. This effect deepens as the context window grows.

Diminishing Returns — Quality vs Utilization

More context does not always mean better output. Curated, relevant context dramatically outperforms a "dump everything" approach.

Curated context

Current utilization

Information Gain (0–35%)

Each additional piece of context adds significant value. The model gains new information it could not infer on its own.

Plateau (35–60%)

Marginal returns flatten. The model has enough information; extra context provides little uplift but some redundancy.

Degradation (60–100%)

Noise overwhelms signal. Contradictions, irrelevant data, and context overload degrade output quality.
