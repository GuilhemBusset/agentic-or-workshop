# Supply Chain Transportation Optimization Suite (5 Problems)

This exercise package contains five supply-chain transportation optimization problems designed for post-workshop agentic coding labs.

## Exercise purpose
- Use different approaches to understand each problem and solve it.
- Focus on building a clear understanding of the problem and its critical points.
- Encourage experimentation: novel approaches are welcome.
- Everyone is free to build what they need to solve the problem.

## Expected outcome
- A robust way (intentionally open-ended) to understand the problem and identify critical points.
- An optimal solution implemented with a production-grade quality.
  - production-grade: Code that is properly split into clear components and covered by tests.

## Design intent
- Problems are hard enough that a one-shot prompt is unlikely to yield a correct, fully feasible model.
- Every problem ships with both a statement and a dataset in its own folder.
